# Розклад Коледжу (College Schedule)

Цей проект є веб-додатком для перегляду розкладу та відмічання пропусків.

## Як запустити

1. Встановіть Node.js (якщо ще не встановлено).
2. Відкрийте термінал у цій папці.
3. Встановіть залежності:
   ```bash
   npm install
   ```
4. Запустіть проект:
   ```bash
   npm run dev
   ```
5. Відкрийте посилання, яке з'явиться в терміналі (зазвичай http://localhost:5173).

## Функціонал

- Перегляд розкладу на тиждень.
- Відмічання пропущених пар (дані зберігаються у вашому браузері).
- Адаптивний дизайн для мобільних телефонів.
